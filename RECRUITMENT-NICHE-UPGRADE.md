# Brownstone Careers — Recruitment Niche Remodel

This package remodels the site around a stronger job-recruiting funnel:

- Homepage repositioned as a remote recruitment gateway with role pathways, candidate fit messaging, process proof, and application CTA.
- Role pages remodeled as a role marketplace with filters, role cards, detailed profiles, skills, responsibilities, and direct apply links.
- Process page converted into a recruiter-style pipeline with candidate preparation guidance and assessment focus areas.
- About page strengthened with recruitment principles, remote readiness, candidate safety, and operating model.
- FAQ, Contact, and Apply pages restyled for clarity, trust, and conversion.
- Header and footer rebuilt for polished logo presentation on dark backgrounds.
- Fully responsive desktop, laptop, tablet, iPad, and mobile layouts.
- Existing Cloudflare Pages Functions and Resend form endpoints preserved.

Deploy using the existing workflow:

```bash
npm install
npm run check
npm run build
npm run pages:deploy
```
