# Private Brownstone Careers Handbook Route

The 22-page Employee & Contractor Handbook has been added as a separate, unlisted static entity.

Production route:

`https://www.brownstonecareers.agency/Brownstone_Careers_Unboarding_Handbook/`

Privacy measures included:

- No navigation links from the public website
- Excluded from sitemap.xml
- `noindex`, `nofollow`, `noarchive`, `nosnippet`, and `noimageindex` directives
- robots.txt disallow rule
- Private/no-store caching headers
- No-referrer policy
- Direct framing disabled

Important: an unlisted URL is not authentication. Anyone who receives or discovers the route can open it. For identity-based restriction, protect this path with Cloudflare Access or another login layer.
