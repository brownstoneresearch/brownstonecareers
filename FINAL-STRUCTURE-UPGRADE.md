# Brownstone Careers — Final Professional Site Upgrade

This package contains the finished high-class recruitment niche version of the Brownstone Careers website.

## Completed upgrades

- Fully polished header and footer for desktop, laptop, iPad/tablet, and mobile screens.
- HD header/footer logo asset added for clearer dark-background brand display.
- Unnecessary role/support photos removed from active pages to create a cleaner, more mature recruitment brand presentation.
- Role cards and detailed role profiles rebuilt into professional pathway panels rather than generic photo cards.
- Homepage, About, Roles, Process, FAQ, Contact, and Apply pages refined for recruiting flow, trust, clarity, and conversion.
- Footer rebuilt with a premium brand panel, candidate CTA, role pathways, support links, and safety notice.
- Mobile layout improved across hero, navigation, role cards, forms, footer, and page covers.
- Existing Cloudflare Pages and Resend form functionality preserved.

## Deployment

Upload the contents of this folder to your GitHub repository or deploy with Cloudflare Pages using the included `dist` folder.
